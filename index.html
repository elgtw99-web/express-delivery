快遞物流狀態查詢系統 示範模式

支援順豐、圓通、中通、申通、韻達、EMS 等 · 聚合平台 API（快遞100）即時查詢

示範模式
試試看： 順豐（已簽收） 圓通（運輸中） EMS（已攬收）
輸入單號並按「查詢物流」即可查看即時物流狀態

🔔 簽收 LINE 通知

按「訂閱此單號」後，系統會把該單號交給快遞100監控；當狀態變為 已簽收時，後端會自動用 LINE Messaging API 推播通知你。
（需先啟動後端並在 .env 設好快遞100與 LINE 金鑰，並關閉「示範模式」。）
