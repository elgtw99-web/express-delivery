# 快遞物流狀態查詢系統 — 設定教學

一個跑在你自己電腦上的小工具：上傳大陸進貨的快遞單號，商品被**簽收**（或退簽／疑難）時，第一時間用 **LINE** 通知你，方便處理後續。

---

## 一、這個資料夾裡有什麼

| 檔案 | 用途 |
|---|---|
| `啟動_快遞查詢系統.bat` | 雙擊即可啟動。會自動裝套件、開啟網頁工具 |
| `app.py` | 網頁小工具主程式 |
| `tracker.py` | 查詢與 LINE 推播引擎 |
| `check_once.py` / `檢查一次.bat` | 只檢查一次（給 Windows 排程用） |
| `requirements.txt` | 需要的套件清單 |
| `config.json` | 你的憑證設定（第一次執行後自動產生） |
| `data.json` | 你的追蹤單號清單（自動產生） |

---

## 二、先裝 Python（只需一次）

1. 到 <https://www.python.org/downloads/> 下載並安裝。
2. **安裝時務必勾選最下面的「Add Python to PATH」**，再按 Install。
3. 裝完就好，不用另外設定。

---

## 三、啟動系統

雙擊 **`啟動_快遞查詢系統.bat`**。第一次會自動安裝套件（約一兩分鐘），然後瀏覽器會自動打開
<http://127.0.0.1:5000>。

> 那個黑色視窗請保持開著，關掉就等於關掉服務。

---

## 四、申請 kuaidi100 查詢憑證（key / customer）

> 免費會員無法直接程式化查詢，需申請「API 實時查詢」，**新用戶有免費體驗額度**。

1. 到 kuaidi100 API 開放平台 <https://api.kuaidi100.com/> 登入（就是你原本的帳號）。
2. 進入「管理後台」→ 找到 **「實時查詢」** 產品，點選開通／免費體驗。
   - 可能需要完成**實名 / 企業認證**才能開通，依畫面指示操作。
3. 開通後，在 **「我的」→「授權碼 / API 資料」** 頁面，就能看到兩組值：
   - **key**（授權 key）
   - **customer**（授權碼 customer）
4. 把這兩組值複製起來，等一下要填。

---

## 五、建立 LINE 推播（Messaging API）

> ⚠️ 舊的「LINE Notify」已於 2025/3/31 停止服務，現在要用 **LINE Messaging API（官方帳號）**，免費方案每月 200 則推播足夠一般用量。

**A. 建立官方帳號頻道，拿 token**
1. 到 LINE Developers <https://developers.line.biz/console/> 用你的 LINE 帳號登入。
2. 建立一個 **Provider**（隨便命名，例如你的店名）。
3. 在該 Provider 底下建立一個 **Messaging API channel**（頻道）。
4. 進入頻道 →「Messaging API」分頁 → 最下方 **Channel access token（long-lived）** 按 Issue 產生，複製這串 token。

**B. 把官方帳號加為好友、取得你的 userId**
1. 同一分頁上方有這個官方帳號的 **QR code / Bot basic ID**，用你手機的 LINE 掃描並**加入好友**（不加好友收不到推播）。
2. 取得 **userId**（U 開頭的一長串）最簡單的方法：
   - 到頻道的「Basic settings」分頁，找 **Your user ID**，即是你自己的 userId；
   - 或用官方的 webhook 工具檢視。填這個 userId，代表推播只發給你自己。

---

## 六、填入設定並測試

1. 在網頁工具右上角點 **「設定」**。
2. 貼上 **key、customer、Channel access token、userId**。
3. 設定「檢查間隔」（建議 30～60 分鐘）與「啟用背景自動檢查＝開啟」，按 **儲存設定**。
4. 按 **「送測試 LINE 推播」**，手機 LINE 有收到訊息就代表成功 ✅。

## 七、日常使用

1. 首頁「新增快遞單號」貼上單號，每行一筆。可帶備註與電話：
   ```
   780123456789
   SF1234567890,A客戶洋裝,0912345678
   773998877665,B批發鞋
   ```
   - 系統會自動辨識快遞公司。
   - **順豐、中通建議填收件電話**，否則可能查不到。
2. 系統會依你設定的間隔自動檢查；也可隨時按 **「立即檢查全部」**。
3. 一旦某筆變成**簽收 / 退簽 / 拒簽 / 疑難**，就會 LINE 通知你一次，並在清單標記「已推播」。

## 八、讓它自動在背景檢查

- **最簡單**：讓 `啟動_快遞查詢系統.bat` 一直開著，它每隔你設定的分鐘數就會自動查一次。
- **開機自動啟動**：在 `啟動_快遞查詢系統.bat` 按右鍵→建立捷徑，把捷徑放進
  `Win+R` 輸入 `shell:startup` 打開的那個資料夾。
- **不想一直開網頁（進階）**：用 Windows「工作排程器」定時執行本資料夾的 `檢查一次.bat`
  （例如每 30 分鐘一次），不開網頁也能查與推播。

## 九、常見問題

- **一定要開著電腦嗎？** 是。查詢與推播都在你電腦上跑，電腦關機或沒網路就不會檢查。
- **是「即時」還是「定時」？** 目前是**定時**（每 N 分鐘查一次），越短越接近即時，但會消耗較多查詢次數。若要做到「簽收當下秒推」需改用 kuaidi100 的「訂閱推送」，那需要一個對外網址接收回呼，之後可再幫你升級。
- **免費額度用完怎麼辦？** kuaidi100 實時查詢超過免費額度後需在後台儲值；可把檢查間隔拉長、單號結案後刪除來省用量。
- **查不到 / 顯示錯誤？** 多半是快遞公司代碼沒辨識出來，或順豐中通沒填電話。可刪除後補上電話重加。
